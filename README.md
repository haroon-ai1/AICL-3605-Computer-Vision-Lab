# AICL 3605 - Computer Vision Lab

Welcome to my repository for the AICL 3605 Computer Vision Lab (Semester 6). This repository contains all weekly lab tasks, scripts, and outputs focusing on image processing, analysis, and computer vision techniques.

## Added Features
Throughout this course, the following major technical approaches and features have been implemented:
* Image enhancement using point processing techniques (contrast enhancement, image negative)
* Edge detection, face detection, and feature extraction (Gabor features)
* Content-based image retrieval and object recognition using Particle Swarm Optimization
* Pattern recognition using PCA, Cluster Analysis, and Artificial Neural Networks
* Advanced segmentation using Grad-CAM visualization and digit/alphabet recognition via Neural Networks

## Navigation Flow
To make grading and reviewing easy, the repository is organized chronologically by week. Navigate to the respective folder to find the source code, input images, and output screenshots for that lab session:

* **Week_01_02_Basics:** Python and OpenCV installation and basic operations
* **Week_03_04_Enhancement:** Scene enhancement, contrast enhancement, and image negative
* **Week_05_Edge_Detection:** Implementations of edge detection methods
* **Week_06_Face_Detection:** Face detection and recognition scripts
* **Week_07_Gabor_Features:** Gabor feature extraction
* **Week_09_Image_Retrieval:** Content-based image retrieval procedures
* **Week_10_11_Object_Detection:** Object detection theory and particle swarm optimization
* **Week_12_13_Pattern_Recognition:** PCA, auto-correlation, and Artificial Neural Networks
* **Week_14_Segmentation:** Image segmentation using Grad-CAM
* **Week_15_Digit_Recognition:** Neural network implementations for character recognition

## How to Run
1. Clone this repository to your local machine.
2. Ensure Python and OpenCV are installed.
3. Navigate to the specific week's folder and execute the provided Python scripts.
